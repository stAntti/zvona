# Видение продукта ZVONA

ZVONA превращает ICP-список в управляемую очередь outreach-задач и подтверждённые квалифицированные outcomes.

```text
ICP import → research → readiness → enrichment → routing
→ outreach task → QA → next-best-action → CRM writeback
```

## Ценность

- Руководитель кампании видит готовность offer и аккаунтов до запуска execution.
- Оператор получает одну task card с evidence, claims, запретами и правилами следующего шага.
- Контролёр качества сверяет outcome с действовавшим task snapshot и транскриптом.
- Sales manager получает только лиды с ЛПР, актуальной потребностью и согласованным следующим шагом.

## Принципы

- Объяснимые правила важнее непрозрачного AI score.
- AI предлагает, но policy gate и человек контролируют внешнее действие.
- Отказ клиента не снижает качество корректной работы оператора.
- Публичный контакт не равен consent.
- Moat строится на outcome data, QA и routing, а не на prompts.

Первая вертикаль использует только выдуманные данные о корпоративных подарках в Казахстане.
