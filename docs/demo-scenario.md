# Демонстрационный сценарий

## Цель

За 5 минут пользователь загружает Excel и проводит один полный демонстрационный звонок без инструкций.

## Путь

`Главная → Запуск кампании → Загрузка Excel → Компании → Карточка компании → Очередь звонков → Звонок → Разбор → Главная`

1. Руководитель нажимает «Продолжить кампанию».
2. Коротко подтверждает предложение и нужные компании.
3. Загружает `.xls`, `.xlsx` или `.csv`. БИН сохраняется строкой.
4. Открывает карточку компании с кратким AI-досье.
5. Добавляет компанию в очередь.
6. Оператор проходит демонстрационный звонок и фиксирует результат.
7. ZVONA показывает оценку, выясненные факты, следующий шаг и начисление.
8. На главной обновляются звонки, найденные ЛПР и квалифицированные лиды.

## Готово, когда

Новый пользователь без объяснений проходит весь путь одной компании за 5 минут.

Демо использует только выдуманные данные, не устанавливает реальный звонок и не сохраняет загруженный файл после обновления страницы.
