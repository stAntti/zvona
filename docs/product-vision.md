# Видение продукта ZVONA

ZVONA — платформа управляемого человеческого B2B-обзвона с AI-копилотом. Она помогает оператору подготовиться, провести разговор по правилам кампании, корректно зафиксировать результат и получить прозрачное начисление за качественную работу.

Первая вертикаль — продажа корпоративных новогодних подарков компаниям Казахстана. Все компании, сотрудники, контакты, разговоры и суммы в прототипе выдуманы.

## Ценность для ролей

- **Оператор** видит приоритетную очередь, контекст компании, подсказки в разговоре, оценку и начисление.
- **Руководитель кампании** видит прогресс кампании и соблюдение правил.
- **Контролёр качества** получает звонки, которые требуют ручной проверки, и апелляции операторов.
- **Менеджер по продажам** получает заинтересованные компании для дальнейшей работы.

## Основной принцип

Оплачивается не сам факт звонка, а валидная работа с учётом качества и результата. AI в прототипе не принимает реальные решения: его поведение воспроизводится детерминированным demo-сценарием.
